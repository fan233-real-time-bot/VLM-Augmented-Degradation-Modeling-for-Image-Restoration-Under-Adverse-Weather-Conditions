### Dataset format:

Download the datasets and arrange them in the following format. 
```
    Dataset
    ├── data 
    |   ├── train # Training  
    |   |   ├── <dataset_name>   
    |   |   |   ├── input         
    |   |   |   └── gt            
    |   |   └── dataset_filename.txt
    |   └── test  # Testing         
    |   |   ├── <dataset_name>          
    |   |   |   ├── input        
    |   |   |   └── gt           
    |   |   └── dataset_filename.txt
```
